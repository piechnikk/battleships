## Getting Started

To get a copy of this project up and running on your local machine, follow these steps:

1. Clone the repository: 
```
git clone https://github.com/piechnikk/battleships.git
```
2. Navigate to the project directory: 
```
cd battleships
```

## Usage

Open the `index.html` file in your preferred web browser to view the website. Place your ships and click "ROZPOCZNIJ ROZGRYWKÄ˜" button to start a game.

### Controls
- right click - rotate the currently selected ship
